## Overview

The Symbol (`zbar_symbol_t`) represents a decoded barcode result. It stores the symbol type, decoded data, quality metrics, and location information.

## Data Types

```c
typedef struct zbar_symbol_s zbar_symbol_t;
typedef struct zbar_symbol_set_s zbar_symbol_set_t;
```

Opaque symbol and symbol set objects.

## Symbol Types

### zbar_symbol_type_t

```c
typedef enum zbar_symbol_type_e {
    ZBAR_NONE        = 0,   // No symbol decoded
    ZBAR_PARTIAL     = 1,   // Intermediate status
    ZBAR_EAN2        = 2,   // GS1 2-digit add-on
    ZBAR_EAN5        = 5,   // GS1 5-digit add-on
    ZBAR_EAN8        = 8,   // EAN-8
    ZBAR_UPCE        = 9,   // UPC-E
    ZBAR_ISBN10      = 10,  // ISBN-10 (from EAN-13)
    ZBAR_UPCA        = 12,  // UPC-A
    ZBAR_EAN13       = 13,  // EAN-13
    ZBAR_ISBN13      = 14,  // ISBN-13 (from EAN-13)
    ZBAR_COMPOSITE   = 15,  // EAN/UPC composite
    ZBAR_I25         = 25,  // Interleaved 2 of 5
    ZBAR_DATABAR     = 34,  // GS1 DataBar (RSS)
    ZBAR_DATABAR_EXP = 35,  // GS1 DataBar Expanded
    ZBAR_CODABAR     = 38,  // Codabar
    ZBAR_CODE39      = 39,  // Code 39
    ZBAR_PDF417      = 57,  // PDF417
    ZBAR_QRCODE      = 64,  // QR Code
    ZBAR_SQCODE      = 80,  // SQ Code
    ZBAR_CODE93      = 93,  // Code 93
    ZBAR_CODE128     = 128, // Code 128
} zbar_symbol_type_t;
```

### zbar_orientation_t

```c
typedef enum zbar_orientation_e {
    ZBAR_ORIENT_UNKNOWN = -1,  // Unable to determine
    ZBAR_ORIENT_UP,            // Upright, read left to right
    ZBAR_ORIENT_RIGHT,         // Sideways, read top to bottom
    ZBAR_ORIENT_DOWN,          // Upside-down, read right to left
    ZBAR_ORIENT_LEFT,          // Sideways, read bottom to top
} zbar_orientation_t;
```

**Since:** 0.11

## Reference Counting

### zbar_symbol_ref()

```c
void zbar_symbol_ref(
    const zbar_symbol_t *symbol,
    int refs
)
```

Manipulates symbol reference count.

<ParamField path="symbol" type="const zbar_symbol_t*" required>
  Symbol object
</ParamField>

<ParamField path="refs" type="int" required>
  Reference count change (+1 to increment, -1 to decrement)
</ParamField>

<Note>
Increment when storing a new reference. Decrement when reference is no longer used. Do not refer to symbol after decrementing if the containing image was destroyed.
</Note>

**Since:** 0.9

**Returns:** Bitmask of configs

**Since:** 0.11

### zbar_symbol_get_modifiers()

```c
unsigned int zbar_symbol_get_modifiers(
    const zbar_symbol_t *symbol
)
```

Retrieves symbology modifier flags (e.g., GS1, AIM).

**Returns:** Bitmask of modifier flags

**Since:** 0.11

### zbar_symbol_get_quality()

```c
int zbar_symbol_get_quality(
    const zbar_symbol_t *symbol
)
```

Retrieves a confidence metric for the symbol.

**Returns:** Unscaled relative quantity where larger values indicate better quality

<Note>
The exact definition may change. Only the relative ordering between values is stable.
</Note>

**Since:** 0.9

### zbar_symbol_get_count()

```c
int zbar_symbol_get_count(
    const zbar_symbol_t *symbol
)
```

Retrieves current cache count. Provides inter-frame reliability for video streams.

**Returns:**
- `< 0` - Symbol still uncertain
- `0` - Symbol newly verified
- `> 0` - Duplicate symbol count

### zbar_symbol_get_orientation()

```c
zbar_orientation_t zbar_symbol_get_orientation(
    const zbar_symbol_t *symbol
)
```

Retrieves general orientation of decoded symbol.

**Returns:** Coarse axis-aligned orientation or `ZBAR_ORIENT_UNKNOWN`

**Since:** 0.11

## Usage Example

```c symbol_example.c
#include <zbar.h>
#include <stdio.h>

int main() {
    zbar_image_scanner_t *scanner = zbar_image_scanner_create();
    zbar_image_t *image = zbar_image_create();
    
    // Setup and scan image...
    int n = zbar_scan_image(scanner, image);
    
    if (n > 0) {
        // Get first symbol
        const zbar_symbol_t *symbol = zbar_image_first_symbol(image);
        
        // Iterate through all symbols
        for (; symbol; symbol = zbar_symbol_next(symbol)) {
            // Get symbol type and data
            zbar_symbol_type_t type = zbar_symbol_get_type(symbol);
            const char *data = zbar_symbol_get_data(symbol);
            unsigned int len = zbar_symbol_get_data_length(symbol);
            
            printf("Type: %s\n", zbar_get_symbol_name(type));
            printf("Data: %.*s\n", len, data);
            printf("Quality: %d\n", zbar_symbol_get_quality(symbol));
            
            // Get location
            unsigned int loc_size = zbar_symbol_get_loc_size(symbol);
            printf("Location points: %u\n", loc_size);
            
            for (unsigned int i = 0; i < loc_size; i++) {
                int x = zbar_symbol_get_loc_x(symbol, i);
                int y = zbar_symbol_get_loc_y(symbol, i);
                printf("  Point %u: (%d, %d)\n", i, x, y);
            }
            
            // Get orientation
            zbar_orientation_t orient = zbar_symbol_get_orientation(symbol);
            printf("Orientation: %s\n", 
                   zbar_get_orientation_name(orient));
            
            // Check for composite
            const zbar_symbol_set_t *components = 
                zbar_symbol_get_components(symbol);
            if (components) {
                int comp_count = zbar_symbol_set_get_size(components);
                printf("Composite with %d components\n", comp_count);
            }
        }
    }
    
    zbar_image_destroy(image);
    zbar_image_scanner_destroy(scanner);
    return 0;
}
```
