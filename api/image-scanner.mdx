## Overview

The Image Scanner (`zbar_image_scanner_t`) provides a mid-level interface for scanning 2D images and detecting barcodes. It processes grayscale images and returns decoded symbol results.

## Data Type

```c
typedef struct zbar_image_scanner_s zbar_image_scanner_t;
```

Opaque image scanner object that maintains state between scans.

## Constructor and Destructor

### zbar_image_scanner_create()

```c

```

Creates a new image scanner instance.

**Returns:** Pointer to new image scanner object.

### zbar_image_scanner_destroy()

```c
void zbar_image_scanner_destroy(zbar_image_scanner_t *scanner)
```

Destroys an image scanner and releases all associated resources.

<ParamField path="scanner" type="zbar_image_scanner_t*" required>
  Scanner instance to destroy
</ParamField>

## Scanning

### zbar_scan_image()

```c
int zbar_scan_image(zbar_image_scanner_t *scanner, zbar_image_t *image)
```

Scans an image for barcodes. The image must be in Y800 or GRAY format.

<ParamField path="scanner" type="zbar_image_scanner_t*" required>
  Scanner instance
</ParamField>

<ParamField path="image" type="zbar_image_t*" required>
  Grayscale image to scan
</ParamField>

**Returns:**
- `> 0` - Number of symbols successfully decoded
- `0` - No symbols found
- `-1` - Error occurred

**Since:** 0.9 (behavior changed to only accept grayscale)

## Result Caching

### zbar_image_scanner_enable_cache()

```c
void zbar_image_scanner_enable_cache(
    zbar_image_scanner_t *scanner,
    int enable
)
```

Enables or disables the inter-image result cache. Useful for video streams to filter duplicate results and add consistency checking.

<ParamField path="scanner" type="zbar_image_scanner_t*" required>
  Scanner instance
</ParamField>

<ParamField path="enable" type="int" required>
  1 to enable, 0 to disable
</ParamField>

<Note>
Enabling the cache clears any existing cached results.
</Note>

### zbar_image_scanner_recycle_image()

```c
void zbar_image_scanner_recycle_image(
    zbar_image_scanner_t *scanner,
    zbar_image_t *image
)
```

Removes previously decoded results from the scanner and image. More efficient than `zbar_image_set_symbols(image, NULL)`.

**Since:** 0.10

## Results

### zbar_image_scanner_get_results()

```c
const zbar_symbol_set_t* zbar_image_scanner_get_results(
    const zbar_image_scanner_t *scanner
)
```

Retrieves decode results for the last scanned image.

**Returns:** Symbol set container or NULL if no results available

<Note>
The symbol set does not have its reference count adjusted. Increment it if you need to keep results after the next scan.
</Note>

**Since:** 0.10

## Usage Example

```c scan_image.c
#include <zbar.h>
#include <stdio.h>

int main() {
    // Create scanner
    zbar_image_scanner_t *scanner = zbar_image_scanner_create();
    
    // Configure to scan QR codes only
    zbar_image_scanner_set_config(scanner, 0, ZBAR_CFG_ENABLE, 0);
    zbar_image_scanner_set_config(scanner, ZBAR_QRCODE, ZBAR_CFG_ENABLE, 1);
    
    // Create and setup image
    zbar_image_t *image = zbar_image_create();
    zbar_image_set_format(image, zbar_fourcc('Y','8','0','0'));
    zbar_image_set_size(image, 640, 480);
    zbar_image_set_data(image, raw_data, width * height, NULL);
    
    // Scan the image
    int n = zbar_scan_image(scanner, image);
    
    if (n > 0) {
        // Extract results
        const zbar_symbol_t *symbol = zbar_image_first_symbol(image);
        for (; symbol; symbol = zbar_symbol_next(symbol)) {
            zbar_symbol_type_t type = zbar_symbol_get_type(symbol);
            const char *data = zbar_symbol_get_data(symbol);
            printf("decoded %s: %s\n", 
                   zbar_get_symbol_name(type), data);
        }
    }
    
    // Cleanup
    zbar_image_destroy(image);
    zbar_image_scanner_destroy(scanner);
    
    return 0;
}
```
