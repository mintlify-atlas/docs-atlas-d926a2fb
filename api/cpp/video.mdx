## Overview

The `zbar::Video` class provides a mid-level abstraction for video capture. It manages video devices and captures images that can be passed to an `ImageScanner` for barcode detection.

**Header:** `<zbar.h>` (includes `zbar/Video.h`)

**Namespace:** `zbar`


</ParamField>

### Video() (with device)


<ParamField path="device" type="std::string&" required>

</ParamField>

**Throws:** Exception if device cannot be opened

## Image Capture

### enable()

```cpp
void enable(bool enable = true)
```

Starts or stops video capture.

<ParamField path="enable" type="bool" default="true">
  True to start capture, false to stop
</ParamField>

**Throws:** Exception on error

**Example:**
```cpp
video.enable(true);   // Start capture
video.enable(false);  // Stop capture
```

### next_image()

```cpp
Image next_image()
```

Retrieves the next captured image from the video stream. This method blocks until an image is available.

<ResponseField name="return" type="Image">
  The next captured image
</ResponseField>

**Throws:** Exception if capture fails or no image available

**Example:**
```cpp
video.enable();
while (true) {
    Image image = video.next_image();
    // Process image...
}
```

## Complete Example

```cpp
#include <iostream>
#include <zbar.h>

using namespace std;
using namespace zbar;

int main() {
    try {
        // Create video capture object
        Video video;
        
        // Request specific resolution before opening
        video.request_size(640, 480);
        
        // Open video device
        string device = "/dev/video0";
        video.open(device);
        
        // Get actual resolution
        cout << "Opened device: " << device << endl;
        cout << "Resolution: " << video.get_width() 
             << "x" << video.get_height() << endl;
        
        // Adjust device controls
        video.set_control("brightness", 128);
        video.set_control("contrast", 32);
        
        // Create scanner
        ImageScanner scanner;
        scanner.set_config(ZBAR_NONE, ZBAR_CFG_ENABLE, 1);
        
        // Start capture
        video.enable();
        
        cout << "Scanning for barcodes..." << endl;
        
        // Capture and scan frames
        while (true) {
            // Get next image
            Image image = video.next_image();
            
            // Scan for barcodes
            int n = scanner.scan(image);
            
            // Process results
            if (n > 0) {
                for (auto symbol = image.symbol_begin();
                     symbol != image.symbol_end(); ++symbol) {
                    cout << "Found: " << symbol->get_type_name() 
                         << " = " << symbol->get_data() << endl;
                    
                    // Exit after first barcode
                    goto done;
                }
            }
        }
        
    done:
        video.enable(false);
        video.close();
        
    } catch (exception &e) {
        cerr << "Error: " << e.what() << endl;
        return 1;
    }
    
    return 0;
}
```

## Platform Support

### Linux
- Uses Video4Linux2 (V4L2) API
- Supports most USB and built-in cameras
- File descriptor available via `get_fd()` for integration with `select()`/`poll()`

### Windows
- Uses DirectShow or VFW
- File descriptor not available

### macOS
- Uses AVFoundation or QTKit
- File descriptor not available
