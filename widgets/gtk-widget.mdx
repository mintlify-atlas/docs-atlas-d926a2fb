## Overview

The `ZBarGtk` widget embeds a barcode reader directly into GTK+ 2.x/3.x based GUI applications. It can process barcodes from a video source (webcam) or from individual `GdkPixbuf` images supplied to `zbar_gtk_scan_image()`.

The widget is implemented as a standard GTK widget using the GObject type system, making it easy to integrate with existing GTK applications and use with GTK Builder/Glade.

<Note>
The GTK widget supports both GTK+ 2.x and GTK+ 3.x, and includes full GObject Introspection (GIR) support for use with Python, JavaScript, and other languages. It is distributed under the GNU Lesser Public License v2.1.
</Note>

## Installation

### Build Requirements

To build applications with the ZBarGtk widget, you need:

- GTK+ 2.x or GTK+ 3.x development libraries
- ZBar library and headers
- pkg-config (for locating libraries)
- For GIR support: libgirepository development packages

<Tip>
GTK+ 3.x is recommended over GTK+ 2.x for better GObject Introspection support and modern GTK features.
</Tip>

### Linking

When building your application, link against the ZBar GTK library:

```bash
# Using pkg-config with GTK+ 3
gcc myapp.c -o myapp $(pkg-config --cflags --libs zbar-gtk)

# Or explicitly with GTK+ 3
gcc myapp.c -o myapp $(pkg-config --cflags --libs gtk+-3.0 zbar)
```

## ZBarGtk Widget Reference

### Header

```c
#include <zbar/zbargtk.h>
```

### Type Macros

```c
#define ZBAR_TYPE_GTK (zbar_gtk_get_type())
#define ZBAR_GTK(object) \
    (G_TYPE_CHECK_INSTANCE_CAST((object), ZBAR_TYPE_GTK, ZBarGtk))
#define ZBAR_GTK_CLASS(klass) \
    (G_TYPE_CHECK_CLASS_CAST((klass), ZBAR_TYPE_GTK, ZBarGtkClass))
#define ZBAR_IS_GTK(object) \
    (G_TYPE_CHECK_INSTANCE_TYPE((object), ZBAR_TYPE_GTK))
```

### Structure Definition

```c
typedef struct _ZBarGtk ZBarGtk;
typedef struct _ZBarGtkClass ZBarGtkClass;

struct _ZBarGtk {
    GtkWidget widget;
    gpointer *_private;
};

struct _ZBarGtkClass {
    GtkWidgetClass parent_class;

    /* Signals */
    void (*decoded)(ZBarGtk *zbar, 
                    zbar_symbol_type_t symbol_type,
                    const char *data);
    
    void (*decoded_text)(ZBarGtk *zbar, 
                         const char *text);
    
    void (*scan_image)(ZBarGtk *zbar, 
                       GdkPixbuf *image);
};
```

## Properties

The ZBarGtk widget exposes the following GObject properties:

### video-device (string)

The currently set video device path.

- Setting a new device opens it and automatically enables video scanning
- Set to `NULL` or empty string `""` to close the current device
- Platform-specific device paths (e.g., `/dev/video0` on Linux)

```c
g_object_set(G_OBJECT(zbar), "video-device", "/dev/video0", NULL);

gchar *device;
g_object_get(G_OBJECT(zbar), "video-device", &device, NULL);
```

### video-enabled (boolean)

Video device streaming state.

- Use to pause/resume video scanning
- Has no effect unless a video device is opened

```c
g_object_set(G_OBJECT(zbar), "video-enabled", TRUE, NULL);
```

### video-opened (boolean, read-only)

Video device opened state.

- `TRUE` if a video device is currently opened
- Errors during streaming will cause the device to be closed

```c
gboolean opened;
g_object_get(G_OBJECT(zbar), "video-opened", &opened, NULL);
```

## Basic Usage

### Video Scanning Application

<Steps>

<Step title="Initialize GTK and create the widget">

```c
#include <gtk/gtk.h>
#include <zbar/zbargtk.h>

int main(int argc, char *argv[])
{
    gtk_init(&argc, &argv);

    // Create main window
    GtkWidget *window = gtk_window_new(GTK_WINDOW_TOPLEVEL);
    gtk_window_set_title(GTK_WINDOW(window), "ZBar Scanner");
    gtk_window_set_default_size(GTK_WINDOW(window), 640, 480);
    
    // Create ZBarGtk widget
    GtkWidget *zbar = zbar_gtk_new();
    gtk_container_add(GTK_CONTAINER(window), zbar);

    g_signal_connect(window, "destroy", G_CALLBACK(gtk_main_quit), NULL);
    
    gtk_widget_show_all(window);
    gtk_main();
    
    return 0;
}
```

</Step>

<Step title="Connect to signals">
Connect to the decoded signals to receive barcode results:

```c
// Callback for decoded signal
static void on_decoded(ZBarGtk *zbar, 
                       zbar_symbol_type_t symbol_type,
                       const char *data, 
                       gpointer user_data)
{
    g_print("Decoded %s: %s\n", 
            zbar_get_symbol_name(symbol_type), 
            data);
}

// Connect the signal
g_signal_connect(zbar, "decoded", G_CALLBACK(on_decoded), NULL);

// Or use the decoded-text signal for combined output
static void on_decoded_text(ZBarGtk *zbar, 
                            const char *text, 
                            gpointer user_data)
{
    g_print("Decoded: %s\n", text);  // Format: "SYMBOL_TYPE:data"
}

g_signal_connect(zbar, "decoded-text", G_CALLBACK(on_decoded_text), NULL);
```

</Step>

<Step title="Open a video device">
Set the video device to start scanning:

```c
// Set via function call
zbar_gtk_set_video_device(ZBAR_GTK(zbar), "/dev/video0");

// Or use GObject property
g_object_set(G_OBJECT(zbar), "video-device", "/dev/video0", NULL);

// Enable video scanning (usually automatic when device is set)
zbar_gtk_set_video_enabled(ZBAR_GTK(zbar), TRUE);
```

</Step>

</Steps>

### Image Scanning

Scan individual images without a video device:

```c
// Load an image using GdkPixbuf
GError *error = NULL;
GdkPixbuf *pixbuf = gdk_pixbuf_new_from_file("barcode.png", &error);

if (pixbuf) {
    // Scan the image
    zbar_gtk_scan_image(ZBAR_GTK(zbar), pixbuf);
    
    // Results will be emitted via the decoded signals
    g_object_unref(pixbuf);
} else {
    g_warning("Failed to load image: %s", error->message);
    g_error_free(error);
}
```

## Configuration Options

### Video Resolution

Request a specific video resolution before opening the device:

```c
// Must be called before the video device is opened
zbar_gtk_request_video_size(ZBAR_GTK(zbar), 1280, 720);

// Then open the device
zbar_gtk_set_video_device(ZBAR_GTK(zbar), "/dev/video0");
```

<Note>
Video size requests must be made before the device is initialized. Setting the size after opening the device will have no effect.
</Note>

### Property Notifications

Monitor property changes using the `notify` signal:

```c
static void on_video_opened(GObject *object, 
                            GParamSpec *param, 
                            gpointer user_data)
{
    gboolean opened;
    g_object_get(object, "video-opened", &opened, NULL);
    
    if (opened) {
        g_print("Video device opened successfully\n");
    } else {
        g_print("Video device closed\n");
    }
}

g_signal_connect(zbar, "notify::video-opened", 
                 G_CALLBACK(on_video_opened), NULL);
```

## Signals

### decoded

```c
void user_function(ZBarGtk *zbar,
                   zbar_symbol_type_t symbol_type,
                   const char *data,
                   gpointer user_data)
```

Emitted when a barcode is decoded from an image. The symbol type and decoded data are provided as separate parameters.

**Parameters:**
- `zbar`: The ZBarGtk widget
- `symbol_type`: The type of symbol decoded (e.g., `ZBAR_QRCODE`, `ZBAR_CODE128`)
- `data`: The decoded data as a null-terminated string
- `user_data`: User data set when the signal handler was connected

### decoded-text

```c
void user_function(ZBarGtk *zbar,
                   const char *text,
                   gpointer user_data)
```

Emitted when a barcode is decoded. The symbol type name is prefixed to the data, separated by a colon.

**Parameters:**
- `zbar`: The ZBarGtk widget
- `text`: The decoded text in format "SYMBOL_TYPE:data"
- `user_data`: User data set when the signal handler was connected

## Complete Example Application

Here's a complete example with video controls and result display:

<CodeGroup>

```c main.c
#include <gtk/gtk.h>
#include <zbar/zbargtk.h>

static GtkWidget *zbar_widget = NULL;
static GtkWidget *status_label = NULL;
static GtkTextView *results_view = NULL;

// Callback when barcode is decoded
static void on_decoded(GtkWidget *widget, 
                       zbar_symbol_type_t symbol_type,
                       const char *data, 
                       gpointer user_data)
{
    GtkTextBuffer *buffer = gtk_text_view_get_buffer(results_view);
    GtkTextIter end;
    
    gtk_text_buffer_get_end_iter(buffer, &end);
    gtk_text_buffer_insert(buffer, &end, 
                          zbar_get_symbol_name(symbol_type), -1);
    gtk_text_buffer_insert(buffer, &end, ": ", -1);
    gtk_text_buffer_insert(buffer, &end, data, -1);
    gtk_text_buffer_insert(buffer, &end, "\n", -1);
    
    // Auto-scroll to the end
    gtk_text_view_scroll_to_iter(results_view, &end, 0.0, FALSE, 0, 0);
}

// Update status when video state changes
static void on_video_opened(GObject *object, 
                            GParamSpec *param, 
                            gpointer user_data)
{
    gboolean opened = zbar_gtk_get_video_opened(ZBAR_GTK(zbar_widget));
    
    gtk_label_set_text(GTK_LABEL(status_label),
                       opened ? "Status: Video opened" : "Status: Video closed");
}

// Video device combo box changed
static void on_device_changed(GtkComboBox *combo, gpointer user_data)
{
    gchar *device = gtk_combo_box_text_get_active_text(GTK_COMBO_BOX_TEXT(combo));
    
    if (device && device[0] != '<') {
        zbar_gtk_set_video_device(ZBAR_GTK(zbar_widget), device);
    } else {
        zbar_gtk_set_video_device(ZBAR_GTK(zbar_widget), NULL);
    }
    
    g_free(device);
}

// Enable/disable button toggled
static void on_enable_toggled(GtkToggleButton *button, gpointer user_data)
{
    gboolean active = gtk_toggle_button_get_active(button);
    zbar_gtk_set_video_enabled(ZBAR_GTK(zbar_widget), active);
}

int main(int argc, char *argv[])
{
    gtk_init(&argc, &argv);

    // Create main window
    GtkWidget *window = gtk_window_new(GTK_WINDOW_TOPLEVEL);
    gtk_window_set_title(GTK_WINDOW(window), "ZBar Barcode Scanner");
    gtk_window_set_default_size(GTK_WINDOW(window), 640, 600);
    gtk_container_set_border_width(GTK_CONTAINER(window), 10);

    // Create vertical box layout
    GtkWidget *vbox = gtk_box_new(GTK_ORIENTATION_VERTICAL, 5);
    gtk_container_add(GTK_CONTAINER(window), vbox);

    // Create ZBarGtk widget
    zbar_widget = zbar_gtk_new();
    gtk_widget_set_size_request(zbar_widget, 640, 480);
    gtk_box_pack_start(GTK_BOX(vbox), zbar_widget, TRUE, TRUE, 0);

    // Device selection
    GtkWidget *device_box = gtk_box_new(GTK_ORIENTATION_HORIZONTAL, 5);
    gtk_box_pack_start(GTK_BOX(vbox), device_box, FALSE, FALSE, 0);
    
    GtkWidget *device_label = gtk_label_new("Video Device:");
    gtk_box_pack_start(GTK_BOX(device_box), device_label, FALSE, FALSE, 0);
    
    GtkWidget *device_combo = gtk_combo_box_text_new();
    gtk_combo_box_text_append_text(GTK_COMBO_BOX_TEXT(device_combo), "/dev/video0");
    gtk_combo_box_text_append_text(GTK_COMBO_BOX_TEXT(device_combo), "/dev/video1");
    gtk_combo_box_text_append_text(GTK_COMBO_BOX_TEXT(device_combo), "<none>");
    gtk_combo_box_set_active(GTK_COMBO_BOX(device_combo), 0);
    gtk_box_pack_start(GTK_BOX(device_box), device_combo, TRUE, TRUE, 0);

    // Enable/disable button
    GtkWidget *enable_button = gtk_toggle_button_new_with_label("Enable Scanning");
    gtk_box_pack_start(GTK_BOX(vbox), enable_button, FALSE, FALSE, 0);

    // Status label
    status_label = gtk_label_new("Status: Not started");
    gtk_box_pack_start(GTK_BOX(vbox), status_label, FALSE, FALSE, 0);

    // Results text view
    GtkWidget *results_scroll = gtk_scrolled_window_new(NULL, NULL);
    gtk_scrolled_window_set_policy(GTK_SCROLLED_WINDOW(results_scroll),
                                   GTK_POLICY_AUTOMATIC, GTK_POLICY_AUTOMATIC);
    gtk_widget_set_size_request(results_scroll, -1, 150);
    gtk_box_pack_start(GTK_BOX(vbox), results_scroll, FALSE, TRUE, 0);
    
    GtkWidget *results_widget = gtk_text_view_new();
    results_view = GTK_TEXT_VIEW(results_widget);
    gtk_text_view_set_editable(results_view, FALSE);
    gtk_text_view_set_wrap_mode(results_view, GTK_WRAP_WORD);
    gtk_container_add(GTK_CONTAINER(results_scroll), results_widget);

    // Connect signals
    g_signal_connect(window, "destroy", G_CALLBACK(gtk_main_quit), NULL);
    g_signal_connect(zbar_widget, "decoded", G_CALLBACK(on_decoded), NULL);
    g_signal_connect(zbar_widget, "notify::video-opened", 
                     G_CALLBACK(on_video_opened), NULL);
    g_signal_connect(device_combo, "changed", 
                     G_CALLBACK(on_device_changed), NULL);
    g_signal_connect(enable_button, "toggled", 
                     G_CALLBACK(on_enable_toggled), NULL);

    // Open initial device
    zbar_gtk_set_video_device(ZBAR_GTK(zbar_widget), "/dev/video0");

    gtk_widget_show_all(window);
    gtk_main();

    return 0;
}
```

## GObject Introspection (GIR) Support

The ZBarGtk widget includes full GObject Introspection support, allowing it to be used from Python, JavaScript, and other languages that support GIR.

### Python Example (PyGObject)

```python
#!/usr/bin/env python3

import gi
gi.require_version('Gtk', '3.0')
gi.require_version('ZBar', '1.0')

from gi.repository import Gtk, ZBar

def on_decoded(zbar, symbol_type, data, user_data):
    print(f"Decoded: {data}")

def main():
    window = Gtk.Window(title="ZBar Scanner")
    window.set_default_size(640, 480)
    window.connect("destroy", Gtk.main_quit)
    
    # Create ZBar widget
    zbar = ZBar.Gtk.new()
    window.add(zbar)
    
    # Connect signal
    zbar.connect("decoded", on_decoded, None)
    
    # Set video device
    zbar.set_property("video-device", "/dev/video0")
    
    window.show_all()
    Gtk.main()

if __name__ == "__main__":
    main()
```

<Tip>
To check if GIR support is available, verify that the typelib file `ZBar-1.0.typelib` is installed (typically in `/usr/lib/girepository-1.0/`).
</Tip>

### JavaScript Example (GJS)

```javascript

const Gtk = imports.gi.Gtk;
```

## Utility Functions

### zbar_gtk_image_from_pixbuf()

Convert a `GdkPixbuf` to a `zbar_image_t` for use with the low-level ZBar API:

```c
GdkPixbuf *pixbuf = gdk_pixbuf_new_from_file("barcode.png", NULL);
zbar_image_t *zimg = zbar_image_create();

if (zbar_gtk_image_from_pixbuf(zimg, pixbuf)) {
    // Successfully converted
    // Now you can use zimg with zbar_scan_image()
    zbar_image_scanner_t *scanner = zbar_image_scanner_create();
    zbar_scan_image(scanner, zimg);
    
    // Process results...
    
    zbar_image_scanner_destroy(scanner);
}

zbar_image_destroy(zimg);
g_object_unref(pixbuf);
```

## Troubleshooting

### Video device fails to open

- Verify the device path is correct (e.g., `/dev/video0` on Linux)
- Check device permissions (user may need to be in the `video` group)
- Ensure the device is not already in use
- Check system logs: `journalctl -f | grep video`

```bash
# Add user to video group
sudo usermod -a -G video $USER

# List available video devices
v4l2-ctl --list-devices
```

### Widget not displaying video

- Ensure the widget is realized (`gtk_widget_realize()`)
- Check that the widget has been added to a container
- Verify `video-enabled` is set to `TRUE`
- On Wayland, ensure proper display backend support

### GIR support not working

Install required packages:

```bash
# Debian/Ubuntu
sudo apt-get install libgirepository1.0-dev gir1.2-gtk-3.0

# Fedora
sudo dnf install gobject-introspection-devel python3-gobject
```

Verify the typelib is installed:

```bash
ls /usr/lib/*/girepository-1.0/ZBar-1.0.typelib
```

### No barcodes detected

- Ensure adequate lighting
- Check that the barcode is in focus
- Try different video resolutions
- Verify barcode symbology is supported

## GTK Builder / Glade Support

The ZBarGtk widget can be used with GTK Builder XML files:

```xml
<?xml version="1.0" encoding="UTF-8"?>
<interface>
  <requires lib="gtk+" version="3.0"/>
  <object class="GtkWindow" id="window">
    <property name="title">ZBar Scanner</property>
    <child>
      <object class="ZBarGtk" id="zbar">
        <property name="visible">True</property>
        <property name="video-device">/dev/video0</property>
        <property name="video-enabled">True</property>
        <signal name="decoded" handler="on_decoded"/>
      </object>
    </child>
  </object>
</interface>
```

Load in your application:

```c
GtkBuilder *builder = gtk_builder_new_from_file("scanner.ui");
GtkWidget *window = GTK_WIDGET(gtk_builder_get_object(builder, "window"));
GtkWidget *zbar = GTK_WIDGET(gtk_builder_get_object(builder, "zbar"));

gtk_builder_connect_signals(builder, NULL);
g_object_unref(builder);

gtk_widget_show_all(window);
```
