## Quick Start

Get up and running with ZBar in minutes:

<Steps>
  <Step title="Install ZBar">
    Install the library and development headers on your system
    
    <CodeGroup>
    ```bash Ubuntu/Debian
    sudo apt-get install libzbar0 libzbar-dev
    ```
    
    ```bash Fedora/RHEL
    sudo dnf install zbar zbar-devel
    ```
    
    ```bash macOS
    brew install zbar
    ```
    
    ```bash From Source
    autoreconf -vfi
    ./configure
    make
    sudo make install
    ```
    </CodeGroup>
  </Step>
  
  <Step title="Scan Your First Barcode">
    Use the command-line tool to decode a barcode image
    
    ```bash
    zbarimg barcode.png
    ```
  </Step>
  
  <Step title="Integrate into Your Application">
    Add barcode scanning to your application using the API
    
    <CodeGroup>
    ```c C API
    #include <zbar.h>
    
    zbar_image_scanner_t *scanner = zbar_image_scanner_create();
    zbar_image_t *image = zbar_image_create();
    
    // Configure and scan...
    int n = zbar_scan_image(scanner, image);
    ```
    
    ```python Python
    import zbar
    
    scanner = zbar.ImageScanner()
    scanner.parse_config('enable')
    
    # Scan image...
    ```
    </CodeGroup>
  </Step>
</Steps>

## Supported Barcode Types

ZBar supports a wide range of 1D and 2D barcode symbologies:

<AccordionGroup>
  <Accordion title="1D Barcodes">
    - **EAN-13/UPC-A** - Standard retail product barcodes
    - **UPC-E** - Compressed version of UPC-A
    - **EAN-8** - Short version of EAN-13
    - **Code 128** - High-density alphanumeric barcodes
    - **Code 93** - Compact alphanumeric symbology
    - **Code 39** - Widely used alphanumeric format
    - **Codabar** - Numeric with special characters
    - **Interleaved 2 of 5** - Numeric-only format
    - **GS1 DataBar** - Retail point-of-sale applications
  </Accordion>
  
  <Accordion title="2D Barcodes">
    - **QR Code** - Popular 2D matrix barcode
    - **SQ Code** - Specialized 2D symbology
  </Accordion>
</AccordionGroup>
